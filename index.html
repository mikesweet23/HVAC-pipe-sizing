How to use the Pipe Sizer

The idea

Each row is one pipe run. You read the flow off your drawing and type it in — the tool picks the correct size, checks it against CIBSE limits, and works out velocity, pressure drop and weight. Rows link into a tree so you can see the whole system and trace the pump's index run.

Build top-down: enter your main first, then work outwards. Every new row automatically feeds from the one above, so the tree builds itself — you only change "Feeds from" on the exceptions.

Step by step

  • Add your main first — click "+ Add pipe", name it (e.g. "Plant → Lynx tee"), leave Feeds from as "Plant / Root".
  • Add branches and terminals — each new row feeds from the one above by default. Change "Feeds from" only where the pipe branches off somewhere else.
  • Type the flow (l/s) — straight off the drawing for that segment.
  • ΔT — set it once; every new row inherits it until you change one.
  • Material — same: set once, repeats until you change it.
  • Length, Fittings %, Equip ΔP — add as you have them.

Seeing the system

Rows indent under their parent, so the structure is visible at a glance — the main at the left, branches stepped in beneath it, terminals stepped in further. Branch rows are shown in bold with a faint blue tint.

Low Loss Header

Tick the LLH box on any row to size that run as a low loss header — it's oversized so velocity drops to the LLH max (default 0.4 m/s, set in the top bar). The correct approach for a primary/secondary decoupler.

Reading the results

The blue columns fill in automatically: recommended Size with actual bore (ID), velocity, Pa/m, segment loss, and weight.

Status: OK means within Pa/m and velocity limits. Review means velocity is outside the band. No fit means no standard size works.

Weights per line

Each row shows three weights: Pipe (bare metalwork), Water (fluid contents), and Design Wt — the filled pipe plus a per-line Buffer % uplift (default 30%) for lagging, cladding and bracketry. Adjust the buffer per row where a run needs more or less. The Design Weight total feeds the tile and the PDF report for structural support and load assessment.

Sizing the pump — the index run

Tick the Index box on each segment along your worst-case route from the plant to the hardest terminal (usually furthest or highest-flow). Those rows highlight, and the Index Σ column shows the head building up cumulatively down the path — so you can watch it climb. The pump panel then adds plant/valve/strainer/static resistances and a safety factor and gives the duty in bar.

Exporting

  • Export CSV — opens in Excel.
  • PDF Report — formal calc sheet with the segment schedule, pump duty and a CIBSE-compliance narrative. Opens in a new tab to save as PDF.

Saving

Your work auto-saves in this browser as you go. It's per-device, so it won't sync between machines — use the exports to share.
