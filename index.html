Typical LLH sizing: velocity 0.1–0.4 m/s · pressure drop <30 Pa/m · header length typically 0.5–2 m

Design Weight = (Pipe + Fluid) × (1 + uplift%). The uplift covers lagging/insulation, cladding, valves, flanges, brackets, and small fittings. Defaults to 20% — increase to 25–30% for plantrooms with dense fittings, or reduce to 10–15% for clean straight runs without insulation. Use the resulting figure for structural support load assessment, lifting plans, and roof load checks.
Thermal Performance & Heat Loss BS EN ISO 12241 · thicknesses to BS 5422:2023
Method: Q/L = ΔT ÷ [ ln(Dₒ/Dᵢ)/(2πλ) + 1/(πDₒh) ]. Insulation thickness is taken from the adi schedule by nearest pipe OD (154 mm → DN150 → 70 mm), overridable per line. Valve jackets use the Dynoteq-derived W/K coefficients (DN25–DN150); values outside that range are extrapolated and marked est. Flange pairs default to 0.40 × the valve coefficient — adjust above once measured data is available. λ is temperature-dependent; the default suits typical LPHW mean temperatures.
Index Run & Pump Sizing based on circuits marked as index run
Tick the Index checkbox on one or more circuits above to calculate the system index pressure and size the pump.
Ticking the index run: tick only the circuits forming one continuous path from the plant to the most onerous terminal. Those are in series, so their losses add. Where the run splits into branches these are in parallel — alternative routes, not additive — so follow only the branch leading to your index terminal and leave the other unticked. The lesser branch is balanced back to suit and does not add to pump head.
